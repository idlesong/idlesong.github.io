2.2.2 emacs common command

  • window
  • C-x o: switch windows
  • C-x 0: close current window
  • Select Copy Paste
  • C-@: Mark Set Start, then move to the end
  • C-w: Cut
  • M-w: Copy
  • C-y: Paste

2.2.3 Dired cheatsheet

  • cursoner move:
  • n: next line
  • p: prev line
  • q: back; ^: parent dir;
  • ent: enter into the subdir
  • i: list subdir below
  • $: hide subdir(toggle)
  • C-u-k: hide subdir forever
  • m: mark current file
  • u: unmark file
  • k: hide marked file
